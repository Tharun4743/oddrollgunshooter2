  • Die has only odd numbers: 1️⃣ 3️⃣ 5️⃣ 7️⃣ 9️⃣
    •                    
  • 1st roll → Face appears
    •                    
  • 2nd roll → Body part appears
    •                    
  • 3rd roll → Gun with 1 bullet
    •                    
  • 4th roll → Gun with 2 bullets
    •                    
  • 5th roll → Gun with 3 bullets (can SHOOT!)
    •                    
  • Shooting disables target's die number box
    •                    
  • Last player standing wins!
